### Instalation
---
Initial, you need pip and python.

1. Clone the repo
   ```sh
   git clone https://github.com/guisteglich/EasyPay.git

2. You need install the requirements:
```shell
pip install requirements.txt
```
3. Run the EasyPay:
```shell
flaks run 
```
